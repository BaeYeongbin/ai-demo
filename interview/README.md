# K-12 AI 학습도구 POC - 인터뷰 활동

K-12 학생을 위한 AI 기반 인터뷰 학습 도구입니다. 선생님이 주제를 입력하면 AI가 적합한 페르소나를 생성하고, 학생은 해당 페르소나를 인터뷰하여 학습 목표를 달성하는 방식으로 학습합니다.

## 주요 기능

### 1. 선생님 모드
- 주제, 과목, 학년 정보 입력
- AI가 자동으로 적합한 페르소나 생성 (1~3개)
- 학습 목표 질문 자동 생성 (기초/확장/도전 난이도)

### 2. 학생 모드 - 인터뷰
- 생성된 페르소나 선택 및 인터뷰
- 페르소나는 근거 기반으로 응답
- 학습 목표 달성을 위한 질문 가이드 제공

### 3. 학생 모드 - 채점
- 인터뷰 내용을 바탕으로 최종 답안 작성
- AI 루브릭 기반 자동 채점 (5개 평가 기준)
- 구체적인 피드백 및 개선 방향 제시

## 설치 방법

### 1. 필수 요구사항
- Python 3.8 이상
- OpenAI API 키 (https://platform.openai.com/api-keys)

### 2. 패키지 설치

```bash
pip install -r requirements.txt
```

### 3. API 키 설정

환경 변수로 OpenAI API 키를 설정해야 합니다.

```bash
# .env 파일 생성
cp .env.example .env

# .env 파일을 열어서 API 키 입력
OPENAI_API_KEY=your_actual_api_key_here
```

또는 직접 환경변수를 설정할 수도 있습니다:
```bash
export OPENAI_API_KEY=your_actual_api_key_here
```

## 실행 방법

```bash
streamlit run app.py
```

브라우저에서 자동으로 `http://localhost:8501` 이 열립니다.

## 사용 방법

### 1단계: 선생님 모드에서 설정
1. "선생님 모드" 탭 선택
2. 주제, 과목, 학년 정보 입력
   - 예: 주제 "일제강점기 문화통치", 과목 "사회", 학년 "중2"
3. "페르소나 및 학습목표 생성" 버튼 클릭
4. 생성된 페르소나와 학습 목표 확인

### 2단계: 학생 모드 - 인터뷰
1. "학생 모드 - 인터뷰" 탭 선택
2. 인터뷰할 페르소나 선택
3. 학습 목표를 참고하여 질문 입력
4. 페르소나의 응답을 읽고 추가 질문 진행
5. 충분한 정보를 수집할 때까지 인터뷰 진행

### 3단계: 학생 모드 - 채점
1. "학생 모드 - 채점" 탭 선택
2. 인터뷰 내용을 바탕으로 최종 답안 작성 (400자 권장)
3. "답안 제출 및 채점" 버튼 클릭
4. 채점 결과 및 피드백 확인
5. 개선 방향을 참고하여 답안 수정 가능

## 평가 기준

AI 루브릭은 다음 5가지 기준으로 답안을 채점합니다:

1. **정확성**: 사실과 개념의 타당성
2. **근거인용**: 출처 명시, 인용과 요약의 구분, 관련성
3. **질문심층**: 원인/결과/대안 등 고차적 탐구
4. **구조·표현**: 논지 전개, 명료성, 용어 적합성
5. **성찰**: 한계 인식, 관점 비교, 다음 탐구 제안

각 기준은 0~3점으로 평가됩니다:
- 0점: 미달
- 1점: 기본
- 2점: 충족
- 3점: 우수

## 기술 스택

- **Frontend**: Streamlit (Python 웹 UI 프레임워크)
- **LLM**: OpenAI GPT-4.1-nano
- **데이터 관리**: Streamlit Session State (메모리 기반)
